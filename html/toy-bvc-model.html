Boundary vector activity

This function g(d) is a nonlinearity (hyperbolic tangent) of distance d from a focal point, which will be used to emulate boundary-vector-cell activity. The focal point will act as a spatial phase or offset for a ring of activity about the same size as the environment. The smoothness of the sigmoid and the focus point (offset) are the parameters that vary between BVCs.

In [4]: 
def bvc(d, smoothness=20 * cm, D_arena=80 * cm, min_margin=0.1):
    return 0.5 + tanh(2 * (d - (1 - min_margin) * D_arena + smoothness) / smoothness) / 2

# Plot bvc(d) for different values of g_radius
x = linspace(0,100,100) * cm

plt.legend(loc='upper left');

    return ax

This function uniformly samples a circular disc for spatial phase offsets.

In [17]: 
def randc(N, radius):
    t = 2 * pi * rand(N)
    r = rand(N) + rand(N)
    r[r > 1] = 2 - r[r > 1]
    r *= radius
    return np.c_[r * cos(t), r * sin(t)].T

sample = randc(1000, 0.9 * R_arena) / cm
scatter(*sample, marker='.', color='b')
axis('equal')
set_arena();

BVC neuron group

These are the equations for the BVC groups of neurons:

In [18]: 
bvc_eqs = """
    f = 0.5 + tanh(2 * (D - (1 - BVC_min) * D_arena + sigma) / sigma) / 2 : 1
    D = sqrt((X - Xp)**2 + (Y - Yp)**2) : cm
    sigma : cm
    Xp : cm # x-dimension spatial offset
    Yp : cm # y-dimension spatial offset
    X : cm
    Y : cm
"""
BVC = NeuronGroup(N_BVC, model=bvc_eqs)

BVC.Xp, BVC.Yp = randc(N_BVC, 0.9 * R_arena)
BVC.sigma = BVC_sigma[0] + diff(BVC_sigma) * rand(N_BVC)
